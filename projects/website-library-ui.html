LIBRARY DATABASE WEBSITE UI

This is a project I am occasionally working on.

Some time ago I became more interested in web development and wanted to deepen my knowledge of it. After building my first portfolio website, I decided to explore different technologies and frameworks.

One of my previous projects involved using Django to create a new portfolio website. This led me to further experiment and learn about Next.js also.

This project is meant as a successor to the Commandline UI for a Library Database that I initially participated in building during my first year of studies.

For this project, I am using Next.js, Tailwind CSS, and TypeScript. I haven't implemented a database yet, but I plan to do so in the future.
